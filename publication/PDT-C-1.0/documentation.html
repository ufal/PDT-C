Documentation

The core documentation for the Prague Dependency Treebank - Consolidated 1.0 (PDT-C 1.0) is available in the form of PDF files listed below; additional documentation is available via external links:

  • Tectogrammatical (deep syntax) annotation manual (in Czech) (in English)
  • Reference version of tectogrammatical annotation manual (in Czech) (in English)
  • Analytical (surface dependency syntax) annotation manual (in Czech) (in English)
  • Morphological annotation manual (in English)
  • Additional semantic annotation: bridging relations, discourse, multiword expressions, etc. (in Czech) (in English)
  • Manual for annotation of discourse relations (in English)
  • Speech reconstruction guide (in Czech) (in English)
  • Additions to tectogrammatical annotation with respect to the PDTSC, PCEDT and Faust (in Czech) (in English)
